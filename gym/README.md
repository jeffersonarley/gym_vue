# Gym App

Proyecto de gym con Vue 3, Vite y Quasar.

## ¿Qué incluye?

- Página principal con grupos musculares: pecho, espalda, pierna y brazos.
- Cada grupo tiene una ruta propia y muestra 4 ejercicios.
- Usa botones e íconos de Quasar.
- Navegación con `vue-router`.

## Ejecutar la app

1. Instala dependencias:

```bash
npm install
```

2. Inicia el servidor de desarrollo:

```bash
npm run dev
```

3. Abre la URL que muestra Vite en tu navegador.

## Producción

Para generar los archivos finales:

```bash
npm run build
```
